Qué es Git

  1. Herramienta que realiza una función del contrli de versiones de código de forma distribuida
  2. Es muy potente
  3. No depende de un repositorio central
  4. Es software libre
  5. Con ella podemos mantener un historial completo de versiones
  6. Podemos movernos, como si tuviéramos un puntero en el tiempo, por todas las revisiones de código y desplazarnos una manera muy ágil.
  7. Es muy rápida
  8. Las ramas pueden tener una línea de progreso diferente de la rama principal

Paso a seguir para utilizar el GitHub en Visual Studio

  1. Entrar en github.com
  2. Clicar en Sign up y registarte
  3. Una vez logeado en tu perfil, creas un nuevo repositorio.
  4. Copias el link de https que acaba en .git
  5. Instalas la Github
  6. Te pedira tu usuario, y despues podras eligir que repositorio quieres clonar para utilizar
  7. A partir de ahí podras abrir el Visual directamente y ver tu repositorio y modificar directamente.
  8. Si necesitas subir cualquier cambio tendras que ir al menu de la izquierda, tercer icono y realizar un commit y un push
